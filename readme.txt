This plugin simplifies the addition of Wanderfly widgets to your WordPress website. You can specify a destination, country, or region to feature on your posts, so your user can explore that area in greater depth on Wanderfly.

== Installation ==

1. Upload `/wanderfly/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your API Key in "Settings" > "Wanderfly"

= How do I get an API Key? =

1. Go to wanderfly.com
2. Create an account if you don't have one already.
3. Once you're logged in, go to the "Wigets" page.
4. In the "Wanderfly Widgets" section, enter your domain name in the text field (eg: http://wanderfly.com), then click "Get Key."
5. After the success message has been displayed, copy the API Key displayed next to your domain name.
6. Go back to your WordPress admin panel, in "Settings" > "Wanderfly."
7. Paste the API Key in the appropriate box.
