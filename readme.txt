This plugin simplify the addition of Wanderfly's widgets to your Wordpress website. It provides an auto-complete used to search for a destination to display into the post edition pages.

== Installation ==

1. Upload `/wanderfly/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your API Key in "Settings" > "Wanderfly"

== Frequently Asked Questions ==

= How do I get an API Key? =

1. Go on wanderfly.com
2. Create an account if you don't have one already.
3. Once you're logged in, go on "My Settings" page.
4. There, go in the "Wanderfly Widgets" section and enter your domain name in the text field (eg: http://wanderfly.com) then click "Get Key" button.
5. After the success message has been displayed, copy your API Key that is displayed right after the domain name you just provided.
6. Paste the API Key into Wanderfly's plugin section of your Wordpress admin panel.
